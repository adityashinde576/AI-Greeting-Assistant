Project: AI Greeting Assistant

A simple Python program to practice input/output, variables, and string formatting.

Features

Takes user's name as input

Takes user's AI interest area as input

Prints a personalized greeting message

How to Run

Make sure you have Python installed.

Save the file as greeting.py.

Run the program:
python greeting.py

Example Output:

Enter your name: Aditya
Enter your area of AI interest (e.g., NLP, CV, Robotics): NLP
Hello Aditya! Great to know you're interested in NLP.
Python will help you master it!
